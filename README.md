## 🏦 Virtual Bank Application

A Full-Stack Banking Application built using:

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

### 🚀 Features Implemented

- Create Account
- Deposit Money
- Withdraw Money
- Transfer Money
- Account Info

### 🔐 Security
- PIN verification
- MongoDB Atlas cloud database
- Environment variables

### 📦 Backend Setup

```bash
cd backend
npm install
npm start
