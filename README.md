# 🏦 Virtual Bank Application

A full-stack virtual banking system built step-by-step using modern web technologies.

## 🚀 Features (Planned)
- Create bank account
- Secure PIN-based access
- Deposit & withdraw money
- Transfer funds
- Search accounts
- Change PIN

## 🛠 Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
